# [Anaxagoras](https://en.wikipedia.org/wiki/Anaxagoras) ([Anaxa](https://honkai-star-rail.fandom.com/wiki/Anaxa))

`anaxagoras` is an experimental library that will generate a valid [Zod](https://zod.dev/) schema and [ATProto Lexicon](https://atproto.com/specs/lexicon) json when defined using the [`typelex`](https://typelex.org/#install) TypeSpec syntax.

This project is highly experimental and subject to change.
